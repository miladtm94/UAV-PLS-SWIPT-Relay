# results/figures/

This directory stores figures generated by the simulation scripts.
Files are **not tracked by git** (see `.gitignore`).

## Expected Output Files

Each simulation script saves a `.fig` (MATLAB figure) and `.png` (300 DPI raster image):

| Filename | Script | Paper Figure |
|----------|--------|-------------|
| `CP_vs_Power.png` | `run_CP_vs_Power.m` | Connection Probability vs Power |
| `CP_vs_PSR.png` | `run_CP_vs_PSR.m` | Connection Probability vs PSR |
| `CP_OPA_vs_EPA.png` | `run_CP_OPA_vs_EPA.m` | CP: OPA vs EPA comparison |
| `IP_vs_Power.png` | `run_IP_vs_Power.m` | Intercept Probability vs Power |
| `ESR_vs_SNR.png` | `run_ESR_vs_SNR.m` | Ergodic Secrecy Rate vs SNR |
| `ESR_vs_PAF.png` | `run_ESR_vs_PAF.m` | ESR vs Power Allocation Factor |
| `SOP_vs_Power.png` | `run_SOP_vs_Power.m` | Secrecy Outage Probability |
| `All_Metrics_vs_PSR.png` | `run_All_Metrics_vs_PSR.m` | All 4 metrics subplot |
| `FlyingUAV_vs_StaticRelay.png` | `run_FlyingUAV_vs_Static.m` | UAV vs static relay |
| `Fig1_CP_vs_Power.png` | `run_Comprehensive_Simulation.m` | Paper Fig. 1 |
| `Fig2_IP_vs_Power.png` | `run_Comprehensive_Simulation.m` | Paper Fig. 2 |
| `Fig3_ESR_Surface.png` | `run_Comprehensive_Simulation.m` | Paper Fig. 3 (3D surface) |
