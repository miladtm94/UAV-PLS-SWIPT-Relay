# data/processed/

This directory contains `.mat` files generated by the simulation scripts.
These files are **not tracked by git** (see `.gitignore`) because they can be
regenerated by running the corresponding scripts.

## Expected Files

| File | Generated By | Description |
|------|-------------|-------------|
| `CP_vs_Power.mat` | `simulations/connection_probability/run_CP_vs_Power.m` | Connection probability vs transmit power data |
| `CP_vs_PSR.mat` | `simulations/connection_probability/run_CP_vs_PSR.m` | Connection probability vs PSR data |
| `ESR_vs_SNR.mat` | `simulations/secrecy_rate/run_ESR_vs_SNR.m` | ESR (simulation + analytical LB) vs SNR |
| `ESR_vs_PSR.mat` | `simulations/comprehensive/run_All_Metrics_vs_PSR.m` | All metrics vs β |
| `Comprehensive_Results.mat` | `simulations/comprehensive/run_Comprehensive_Simulation.m` | All main paper figures data |

## How to Regenerate

```matlab
% From the MATLAB command window (from repository root):
cd simulations/connection_probability
run_CP_vs_Power          % saves CP_vs_Power.mat

cd ../secrecy_rate
run_ESR_vs_SNR           % saves ESR_vs_SNR.mat

cd ../comprehensive
run_Comprehensive_Simulation  % saves Comprehensive_Results.mat
```

Each script automatically creates this directory if it does not exist and saves
its numerical results here for later use (e.g., for plotting without re-running).
